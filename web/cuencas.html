Información
Actualmente te encuentras en la tabla Cuencas Hidrográficas, esta es una de las dos tablas secundarias que tiene la Base de Datos, en esta podrás encontrar las distintas cuencas Hidrográficas donde se realizan actividades de Pesca en el país.

A diferencia de la tabla Pescas, en esta solo se específica el nombre del método, ningún otro dato es necesario.
Ayuda
En esta página puedes realizar 3 acciones distintas sobre la tabla Cuencas Hidrográficas, las acciones son: Agregar una nueva Cuenca, Editar una Cuenca existente o Eliminar una Cuenca existente. Estas operaciones las puedes realizar en las tarjetas que se encuentran en la parte derecha, a continuación se explicará como realizar cada una de las operaciones:

Para añadir una nueva Cuenca debes simplemente ingresar el nombre y presionar el símbolo +.

Para editar una Cuenca existente primero debes seleccionar la Cuenca que deseas modificar, luego debes llenar el campo con el nuevo nombre para la Cuenca y luego presionar el símbolo del lápiz.

Para eliminar solamente debes seleccionar la Cuenca que deseas eliminar y presionar el símbolo de la basura. CUIDADO: Esta acción no tiene vuelta atras.
¿Qué es esto?
Esto no funciona jsjsjs, pero si quieres cerrar la aplicación presiona la combinación de teclas: Ctrl + w
